# CRM Lead Automation (n8n)

## Overview
This project automates lead collection and qualification using n8n.

<img width="1413" height="965" alt="architecture" src="https://github.com/user-attachments/assets/1aa535a0-815c-4887-b63c-d1d4f5231738" />

## Features
- Form submission trigger
- Lead scoring system
- Google Sheets CRM storage
- Automated email responses
- Slack sales notifications

## Workflow
Form → Lead Scoring → CRM → Qualification → Notifications

## Tech Stack
- n8n
- Google Sheets
- Gmail API
- Slack API

## Setup
1. Import workflow.json into n8n
2. Add your own credentials
3. Activate workflow
